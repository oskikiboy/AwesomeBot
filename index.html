About

The principles behind the bot.

Awesome Features

Our bot offers a plethora of genuinely useful, robust commands for just about anything you could want on Discord. This is how bots were always meant to be: easy to use, integrated with the platform, and connected to the web.

Customizable and Extensible

AwesomeBot's comprehensive control panel allows server admins to use the bot's moderation tools, individually configure features, and add custom content. We also support a powerful extension system that allows developers to addon to the bot.

Free and Open-Source

We believe a Discord bot should be free, forever. That's why AwesomeBot is hosted on a public server with reliable uptime and frequent updates. For tinkerers and developers, we've fully open-sourced the code under the CCPL.

Configurability

You can change everything.

With the AwesomeBot admin console, there's nothing a server owner can't change about the bot. From adding other admins to managing strikes for users to joined/left messages to building an extension, we've included everything. This is how our bot adapts to each and every server it joins.

Polls and Trivia

Live and completely in-chat

Have you ever wanted to do a quick poll in the server? Sure, you can use Strawpoll, but you'd have to leave Discord and post a link. It's not integrated into the Discord experience. With AwesomeBot, we've built thoughtfully integrated polls with live results and easy setup.

In the days of IRC, members of a server would play fun trivia games together. We're bringing that experience back with AwesomeBot. An exhaustive, Jeopardy-style trivia set is included by default with support for custom trivia sets in the JSON syntax. It's very easy to start a game, and users are awarded AwesomePoints for correct answers. It's fun for the whole server!

Full-Featured Stats System

Data for every server.

Our full-featured stats system logs useful usage data for every server about popular games, member activity, and command usage. They can be viewed at any time with the stats command and are reset weekly.

Robust Moderation

Awesome for mods.

With AwesomeBot, moderating a server is no longer a chore. We've included tons of high-quality, easy-to-use tools to make the lives of mods easy. Built-in kick, ban, and mute commands that can be restricted to admins allow modding on mobile. A strikes system for mods to keep track of misbehaving users eliminates the need to store that info elsewhere. Filters for spam and NSFW content automatically keep the chat clean. Optional ban and unban messages allow mods to alert users of their actions. And there's so much more: try AwesomeBot today to get started!
